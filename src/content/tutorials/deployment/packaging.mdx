# Deployment

Welcome to the Deployment module! In this advanced tutorial, you'll learn how to deploy MCP servers from your local machine to production environments using various deployment strategies. We'll cover everything from local development setups to cloud deployments with monitoring and security considerations.

## What You'll Learn

In this module, we'll cover:

- **Local Options**: Different ways to run MCP servers locally (stdio, HTTP, Docker)
- **Docker Deployment**: Containerizing MCP servers with Docker and Docker Compose
- **Azure Deployment**: Full deployment pipeline to Azure Container Apps
- **VPS Deployment**: Setting up MCP servers on a VPS with nginx and systemd
- **Production Deployment**: Scaling, monitoring, security, and best practices

Before starting this module, ensure you have:

✅ **Basic knowledge** of Docker and containerization
✅ **Azure account** (for Azure deployment)
✅ **Linux VPS** (for VPS deployment)
