# MCP Prompts

## Introduction to Prompts

**Prompts** in MCP are reusable templates that define how AI systems should interact with users and respond to specific scenarios. While Resources expose data and Tools enable actions, Prompts provide structured conversation templates that ensure consistent, high-quality interactions across different AI platforms.

### What is an MCP Prompt?

An MCP Prompt is a structured template that:

- Defines conversation flows
- Provides contextual information
- Includes response guidelines
- Manages multi-turn interactions
- Ensures consistency across AI platforms

```typescript
// Example Prompt definition
const prompt = {
  name: "code-review",
  description: "Review code for best practices and potential issues",
  template: `You are an expert code reviewer. Please review the following code:

\`\`\`{{language}}
{{code}}
\`\`\`

Please provide:
1. Overall code quality assessment
2. Specific issues found
3. Improvement suggestions
4. Security considerations

Format your response in markdown with clear sections.`,
  variables: ["language", "code"],
  context: {
    programmingLanguages: ["JavaScript", "Python", "TypeScript", "Java"]
  }
};
```

## Prompt Architecture

### Prompt Components

```typescript
interface Prompt {
  name: string;                    // Unique identifier
  description: string;           // Human-readable description
  template: string;              // Template content with variables
  variables: string[];          // Template variables that can be substituted
  context?: PromptContext;       // Additional context information
  constraints?: PromptConstraints; // Response constraints
}

interface PromptContext {
  // Additional information that helps the AI understand the scenario
  examples?: Example[];
  guidelines?: string[];
  rules?: string[];
}

interface PromptConstraints {
  // Constraints on the AI's response
  maxLength?: number;
  format?: "text" | "markdown" | "json";
  tone?: "formal" | "casual" | "technical";
  requiredSections?: string[];
}
```

## Types of Prompts

### 1. Static Prompts

Static prompts have fixed content without variables:

```typescript
const staticPrompt = {
  name: "welcome-message",
  description: "Welcome message for new users",
  template: `Welcome to our application! Thank you for choosing our service.

Here's what you can do next:
1. Complete your profile setup
2. Explore the documentation
3. Try some example prompts
4. Contact support if you need help

Is there anything specific you'd like to know about?`
};
```

**Use Cases:**
- Welcome messages
- Error responses
- Help text
- Static instructions

### 2. Dynamic Prompts

Dynamic prompts use variables to generate context-specific responses:

```typescript
const dynamicPrompt = {
  name: "generate-report",
  description: "Generate a comprehensive report",
  template: `Generate a {{reportType}} report for {{timePeriod}} based on the following data:

{{dataSummary}}

- Performance indicators
- Trends analysis
- Recommendations
- Next steps

Use {{tone}} tone and format as {{format}}.`,
  variables: ["reportType", "timePeriod", "dataSummary", "tone", "format"],
  constraints: {
    format: "markdown",
    maxLength: 2000,
    requiredSections: ["Overview", "Metrics", "Recommendations"]
  }
};
```

**Use Cases:**
- Report generation
- Personalized responses
- Context-aware advice
- Customized documentation

### 3. Multi-Turn Prompts

Multi-turn prompts manage conversational flows:

```typescript
const multiTurnPrompt = {
  name: "customer-support",
  description: "Customer support conversation flow",
  turns: [

      step: 4,

    }

```

**Use Cases:**
- Customer support workflows
- Multi-step wizards
- Interview processes
- Educational interactions

### 4. Conditional Prompts

Conditional prompts adapt based on user context or input:

```typescript
const conditionalPrompt = {
  name: "code-generation",
  description: "Generate code based on user requirements",
  template: "{{#if userLevel === 'beginner'}}

{{/if}}
```

- Adaptive tutorials
- Role-based responses
- Context-aware help
- Progressive disclosure

## Implementing Prompts

### Basic Prompt Server

```typescript
import { MCPServer } from "@modelcontextprotocol/sdk/server/index.js";

class PromptServer extends MCPServer {
  private prompts = new Map<string, Prompt>();

  constructor() {
    super({
      name: "prompt-server",
      version: "1.0.0"
    });

    // Register prompts
    this.registerPrompt({
      name: "code-review",
      description: "Review code for best practices and potential issues",
      template: `You are an expert code reviewer. Please review the following code:

\`\`\`{{language}}
{{code}}
\`\`\`

Please provide:
1. Overall code quality assessment
2. Specific issues found
3. Improvement suggestions
4. Security considerations

Format your response in markdown with clear sections.`,
      variables: ["language", "code"]
    });

    this.registerPrompt({
      name: "debug-assistance",
      description: "Help debug technical problems",
      template: `I need help debugging a {{problemType}} issue. Here are the details:

{{context}}

The error message is: {{errorMessage}}

What steps should I take to resolve this issue?`,
      variables: ["problemType", "context", "errorMessage"]
    });
  }

  async getPrompt(name: string, variables?: Record<string, any>): Promise<PromptResult> {
    const prompt = this.prompts.get(name);
    if (!prompt) {
      throw new Error(`Prompt not found: ${name}`);
    }

    // Substitute variables if provided
    const template = this.substituteVariables(prompt.template, variables || {});

    return {
      name,
      description: prompt.description,
      template,
      variables: prompt.variables,
      timestamp: new Date().toISOString()
    };
  }

  private substituteVariables(template: string, variables: Record<string, any>): string {
    return template.replace(/\{\{(\w+)\}\}/g, (match, varName) => {
      return variables[varName] !== undefined ? variables[varName] : match;
    });
  }
}
```

### Advanced Prompt Management

```typescript
class AdvancedPromptServer extends MCPServer {
  constructor() {
    super({
      name: "advanced-prompts",
      version: "1.0.0"
    });

    this.setupPromptHierarchy();
    this.setupPromptCache();
    this.setupVersionControl();
  }

  private setupPromptHierarchy() {
    // Organize prompts in a hierarchical structure
    this.promptCategories = {
      "development": {
        "code-review": this.createCodeReviewPrompt(),
        "debug-assistance": this.createDebugPrompt(),
        "documentation": this.createDocumentationPrompt()
      },
      "business": {
        "report-generation": this.createReportPrompt(),
        "analysis": this.createAnalysisPrompt(),
        "presentation": this.createPresentationPrompt()
      },
      "general": {
        "help": this.createHelpPrompt(),
        "feedback": this.createFeedbackPrompt()
      }
    };
  }

  async getPromptByCategory(category: string, name: string, variables?: Record<string, any>): Promise<PromptResult> {
    const categoryPrompts = this.promptCategories[category];
    if (!categoryPrompts) {
      throw new Error(`Category not found: ${category}`);
    }

    const prompt = categoryPrompts[name];
    if (!prompt) {
      throw new Error(`Prompt not found: ${category}.${name}`);
    }

    // Apply version control if needed
    const versionedPrompt = this.getVersionedPrompt(prompt, variables?.version);

    // Apply constraints
    this.validatePromptConstraints(versionedPrompt, variables);

    return {
      ...versionedPrompt,
      template: this.substituteVariables(versionedPrompt.template, variables || {})
    };
  }

  private getVersionedPrompt(prompt: Prompt, version?: string): Prompt {
    // Implement version control logic
    if (version) {
      return this.promptVersions.get(`${prompt.name}@${version}`) || prompt;
    }
    return prompt;
  }
}
```

```typescript
class HandlebarsPromptServer extends MCPServer {
  private handlebars = require('handlebars');

  constructor() {
    super({
      name: "handlebars-prompts",
      version: "1.0.0"
    });

    // Register helpers
    this.handlebars.registerHelper('json', (context) => {
      return JSON.stringify(context, null, 2);
    });

    this.handlebars.registerHelper('formatDate', (date) => {
      return new Date(date).toLocaleDateString();
    });
  }

  async renderPrompt(name: string, context: Record<string, any>): Promise<string> {
    const prompt = this.prompts.get(name);
    if (!prompt) {
      throw new Error(`Prompt not found: ${name}`);
    }

    // Compile template
    const compiled = this.handlebars.compile(prompt.template);

    // Render with context
    return compiled(context);
  }
}
```

## Prompt Security and Best Practices

### Input Sanitation

```typescript
class SecurePromptServer extends MCPServer {
  async renderPrompt(name: string, context: Record<string, any>): Promise<string> {
    const prompt = this.prompts.get(name);
    if (!prompt) {
      throw new Error(`Prompt not found: ${name}`);
    }

    // Sanitize context inputs
    const sanitizedContext = this.sanitizeContext(context);

    // Validate template safety
    this.validateTemplateSafety(prompt.template);

    // Render prompt
    return this.substituteVariables(prompt.template, sanitizedContext);
  }

  private sanitizeContext(context: Record<string, any>): Record<string, any> {
    const sanitized = {};

    for (const [key, value] of Object.entries(context)) {
      if (typeof value === "string") {
        // Escape HTML characters
        sanitized[key] = this.escapeHtml(value);
        // Remove potentially dangerous patterns
        sanitized[key] = this.removeDangerousPatterns(sanitized[key]);
      } else {
        sanitized[key] = value;
      }
    }

    return sanitized;
  }

  private escapeHtml(text: string): string {
    return text
      .replace(/&/g, "&amp;")
      .replace(/</g, "&lt;")
      .replace(/>/g, "&gt;")
      .replace(/"/g, "&quot;")
      .replace(/'/g, "&#039;");
  }

  private removeDangerousPatterns(text: string): string {
    // Remove potentially dangerous template patterns
    return text
      .replace(/\{\{.*?\}\}/g, "") // Remove template injections
      .replace(/<script\b[^<]*(?:(?!<\/script>)<[^<]*)*<\/script>/gi, "") // Remove script tags
      .replace(/javascript:/gi, ""); // Remove javascript protocol
  }
}
```

### Prompt Testing

```typescript
class TestablePromptServer extends MCPServer {
  private testCases = new Map<string, PromptTestCase[]>();

  addPromptTest(promptName: string, test: PromptTestCase) {
    if (!this.testCases.has(promptName)) {
      this.testCases.set(promptName, []);
    }
    this.testCases.get(promptName)!.push(test);
  }

  async runPromptTests(promptName: string): Promise<TestResult[]> {
    const tests = this.testCases.get(promptName) || [];
    const results: TestResult[] = [];

    for (const test of tests) {
      try {
        const result = await this.renderPrompt(promptName, test.context);
        const passed = this.validateResult(result, test.expectedOutput);

        results.push({
          test,
          passed,
          result,
          error: passed ? null : "Output validation failed"
        });
      } catch (error) {
        results.push({
          test,
          passed: false,
          result: null,
          error: error.message
        });
      }
    }

    return results;
  }

  private validateResult(actual: string, expected: string | TestValidator): boolean {
    if (typeof expected === "string") {
      return actual.includes(expected);
    }

    // Handle custom validators
    if (expected.type === "regex") {
      return new RegExp(expected.pattern).test(actual);
    }

    if (expected.type === "length") {
      return actual.length >= expected.minLength && actual.length <= expected.maxLength;
    }

    return false;
  }
}
```
