# MCP Tools

## Introduction to Tools

**Tools** are the action-oriented components of MCP that enable AI systems to perform specific operations and execute functions. While Resources expose data, Tools enable AI models to "do" things - read files, write data, make API calls, and perform any action that your server implements.

### What is a Tool?

A Tool in MCP is a well-defined function that an AI system can call with specific parameters. Each Tool has:

- **Name**: Unique identifier for the tool
- **Description**: Human-readable explanation of what the tool does
- **Input Schema**: Definition of required and optional parameters
- **Implementation**: The actual function logic

```typescript
// Example Tool definition
const tool = {
  name: "calculate-sum",
  description: "Calculate the sum of two numbers",
  inputSchema: {
    type: "object",
    properties: {
      a: { type: "number", description: "First number" },
      b: { type: "number", description: "Second number" }
    },
    required: ["a", "b"]
  },
  async execute({ a, b }) {
    return { result: a + b };
  }
};
```

## Tool Architecture

### Tool Components

```typescript
interface Tool {
  name: string;                    // Unique identifier
  description: string;           // Human-readable description
  inputSchema: JSONSchema;       // Input parameter schema
  outputSchema?: JSONSchema;     // Expected output structure (optional)
  execute: ToolFunction;         // Implementation function
}

type ToolFunction = (args: any) => Promise<ToolResult>;
```

### Tool Result Format

All Tools return results in a standardized format:

```typescript
interface ToolResult {
  content: ToolContent[];        // Response content
  isError?: boolean;             // Error flag
  isErrorDetailed?: string;      // Detailed error message
}
```

### 4. Processing Tools

Tools that perform data processing or transformations:

```typescript
// Data processing tool
const processCsvTool: Tool = {
  name: "process-csv",
  description: "Process and analyze CSV data",
  inputSchema: {
    type: "object",
    properties: {
      csvData: {
        type: "string",
        description: "CSV data as string"
      },
      operation: {
        type: "string",
        enum: ["sum", "average", "count", "group"],
        description: "Processing operation to perform"
      },
      column: {
        type: "string",
        description: "Column to operate on (for sum/average/count)"
      }
    },
    required: ["csvData", "operation"]
  },
  async execute({ csvData, operation, column }) {
    try {
      const rows = parseCSV(csvData);
      let result;

      switch (operation) {
        case "sum":
          const values = rows.map(row => parseFloat(row[column])).filter(v => !isNaN(v));
          result = { sum: values.reduce((a, b) => a + b, 0) };
          break;
        case "average":
          const vals = rows.map(row => parseFloat(row[column])).filter(v => !isNaN(v));
          result = { average: vals.reduce((a, b) => a + b, 0) / vals.length };
          break;
        case "count":
          result = { count: rows.length };
          break;
        case "group":
          const groups = rows.reduce((acc, row) => {
            const key = row[column];
            acc[key] = (acc[key] || 0) + 1;
            return acc;
          }, {});
          result = groups;
          break;
      }

      return {
        content: [{
          type: "json",
          json: result
        }]
      };
    } catch (error) {
      return {
        content: [{
          type: "text",
          text: `CSV processing error: ${error.message}`
        }],
        isError: true,
        isErrorDetailed: error.message
      };
    }
  }
};
```

## Tool Security Considerations

### Input Validation

```typescript
class SecureToolServer extends MCPServer {
  async executeTool(toolName: string, args: any): Promise<ToolResult> {
    const tool = this.tools.get(toolName);
    if (!tool) {
      throw new Error(`Tool not found: ${toolName}`);
    }

    // Validate input schema
    const validationResult = this.validateInput(tool.inputSchema, args);
    if (!validationResult.valid) {
      return {
        content: [{
          type: "text",
          text: `Input validation failed: ${validationResult.error}`
        }],
        isError: true,
        isErrorDetailed: validationResult.error
      };
    }

    // Sanitize inputs for security
    const sanitizedArgs = this.sanitizeInputs(args, tool);

    // Execute tool with sanitized arguments
    return tool.execute(sanitizedArgs);
  }

  private validateInput(schema: JSONSchema, input: any): ValidationResult {
    // Implement JSON Schema validation
    // This prevents injection attacks and ensures correct input types
    try {
      const ajv = new AJV();
      const validate = ajv.compile(schema);
      const valid = validate(input);

      if (!valid) {
        return {
          valid: false,
          error: validate.errors.map(err => err.message).join(", ")
        };
      }

      return { valid: true };
    } catch (error) {
      return {
        valid: false,
        error: `Schema validation error: ${error.message}`
      };
    }
  }

  private sanitizeInputs(args: any, tool: Tool): any {
    // Sanitize inputs based on tool type
    // Remove potentially dangerous properties, escape strings, etc.
    const sanitized = { ...args };

    // Example: Remove null prototype objects
    if (args && args.__proto__ === null) {
      sanitized.__proto__ = Object.prototype;
    }

    // Example: Escape HTML in string inputs
    Object.keys(sanitized).forEach(key => {
      if (typeof sanitized[key] === 'string') {
        sanitized[key] = this.escapeHtml(sanitized[key]);
      }
    });

    return sanitized;
  }
}
```

### Rate Limiting

```typescript
import { RateLimiter } from 'limiter';

class RateLimitedToolServer extends MCPServer {
  private limiter = new RateLimiter(10, 60000); // 10 requests per minute
  private toolLimits = new Map<string, RateLimiter>();

  constructor() {
    super({ name: "rate-limited-tools", version: "1.0.0" });
    this.setupToolLimits();
  }

  private setupToolLimits() {
    // Different limits for different tools
    this.toolLimits.set("heavy-operation", new RateLimiter(1, 30000)); // 1 per 30 seconds
    this.toolLimits.set("api-call", new RateLimiter(5, 60000));     // 5 per minute
    this.toolLimits.set("file-read", new RateLimiter(20, 60000));    // 20 per minute
  }

  async executeTool(toolName: string, args: any): Promise<ToolResult> {
    // Check global rate limit
    try {
      await this.limiter.removeTokens(1);
    } catch (error) {
      return {
        content: [{
          type: "text",
          text: "Rate limit exceeded. Please try again later."
        }],
        isError: true
      };
    }

    // Check tool-specific rate limit
    const toolLimiter = this.toolLimits.get(toolName);
    if (toolLimiter) {
      try {
        await toolLimiter.removeTokens(1);
      } catch (error) {
        return {
          content: [{
            type: "text",
            text: `Rate limit exceeded for ${toolName}. Please try again later.`
          }],
          isError: true
        };
      }
    }

    return super.executeTool(toolName, args);
  }
}
```

## Tool Composition and Chaining

### Sequential Execution

```typescript
class ChainedToolServer extends MCPServer {
  async executeWorkflow(workflow: WorkflowStep[]): Promise<ToolResult> {
    const results = [];
    let lastResult = null;

    for (const step of workflow) {
      // Pass previous result to next step if needed
      const args = step.input ?
        this.parseInputTemplate(step.input, lastResult) :
        {};

      const result = await this.executeTool(step.tool, args);
      results.push(result);
      lastResult = result;
    }

    return {
      content: [{
        type: "json",
        json: {
          workflow: workflow.map((step, i) => ({
            step: i + 1,
            tool: step.tool,
            result: results[i]
          }))
        }
      }]
    };
  }

  private parseInputTemplate(template: string, context: any): any {
    // Parse template strings like "Use this data: {{result.data}}"
    return JSON.parse(template.replace(/\{\{(.+?)\}\}/g, (match, path) => {
      return this.getNestedValue(context, path);
    }));
  }

  private getNestedValue(obj: any, path: string): any {
    return path.split('.').reduce((current, key) => current?.[key], obj);
  }
}
```

### Parallel Execution

```typescript
class ParallelToolServer extends MCPServer {
  async executeParallel(tools: Array<{ name: string; args: any }>): Promise<ToolResult> {
    const promises = tools.map(tool =>
      this.executeTool(tool.name, tool.args)
        .catch(error => ({
          content: [{
            type: "text",
            text: `Error executing ${tool.name}: ${error.message}`
          }],
          isError: true,
          isErrorDetailed: error.message
        }))
    );

    const results = await Promise.all(promises);

    return {
      content: [{
        type: "json",
        json: {
          executed: tools.length,
          results: results.map((result, i) => ({
            tool: tools[i].name,
            success: !result.isError,
            data: result.content[0]
          }))
        }
      }]
    };
  }
}
```
